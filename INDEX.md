1. **Run the program:**

3. **Select option 2** to view all contacts

4. **Explore other options** from the menu

That's it! Your contacts are automatically saved to `contacts.json`.

## 📦 What Each File Does

### contact_management_system.py
- **Main application file**
- Contains Contact class and ContactManager class
- Includes interactive user interface
- Auto-creates and manages contacts.json
- Run this to use the application

### contacts.json (Auto-created)
- **Your contact database**
- JSON format with phone as key
- Auto-saved on every change
- Portable - can be moved/backed up
- Format: `{"phone": {"name": "...", "email": "...", ...}}`
